3. 使用本地yum¶

这篇我们讲个最简单的本地yum源。

添加光盘的步骤我们略过，本环境系统下已存在ISO镜像

本实验在centos7下进行。

3.1. 创建用于挂在光盘的目录¶

mkdir /mnt/iso

3.2. 挂在光盘到本地目录¶

mount /dev/cdrom /mnt/iso

3.3. 创建yum仓库¶

echo "
[base]
name=localiso
baseurl=file:///mnt/iso
gpgcheck=0
enable=1" > /etc/yum.repos.d/local.repo

3.4. 清空yum缓存¶

yum clean all

3.5. 查看yum源列表¶

yum repolist
