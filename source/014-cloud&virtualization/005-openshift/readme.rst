OpenShift到底是个啥？

从架构角度，用一句话来形容OpenShift，那它就是企业版的Kuberrnetes

从功能角度，用一句话来说OpenShift，那它就是下一代应用承载平台

从面向对象角度，用一句话来说OpenShift，那它是“同时面向运维和开发的企业级PaaS平台“。面向运维体现的是容器云，面向开发体现的是DevOps
