安装docker
===================


.. code-block:: bash

    $ sudo wget -P /etc/yum.repos.d/ https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    $ sudo yum install docker-ce-17.12.1.ce
