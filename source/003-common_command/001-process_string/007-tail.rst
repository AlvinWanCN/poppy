tail
####


查看文件alvin.log的最后三行
=======================================

.. code-block:: bash

    $ tail -3 alvin.log


实时打印文件alvin.log的内容
===================================
实时打印文件alvin.log的内容，有新的内容后立即输出

.. code-block:: bash

    $ tail -f alvin.log

实时打印文件alvin.log的内容，从最后20行开始打印
=======================================================


.. code-block:: bash

    $ tail -20f alvin.log
