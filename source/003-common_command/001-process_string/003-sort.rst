sort
###########

Description
====================

-n 按数字排序

-k 指定按照第几列进行排序

-r 倒序排序，数字大的在前面



指定第九列按照数字顺序排列
========================================
.. code-block:: bash

    sort -n -k 9 file.txt

指定第九列按照数字倒序排列
========================================
-r 倒叙排列，数字大的在前面。

.. code-block:: bash

    sort -nr -k 9 file.txt
