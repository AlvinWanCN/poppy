finger
##########

查看用户信息的一个工具

安装finger
===================
如果没有安装需要先安装

.. code-block:: bash

    # yum install finger -y

查看用户ooo的信息
======================

.. code-block:: bash

    [root@alvin ~]# finger ooo
    Login: ooo            			Name:
    Directory: /home/ooo                	Shell: /bin/bash
    Last login Thu Aug  9 12:37 (CST) on pts/0
    No mail.
    No Plan.
    [root@alvin ~]
