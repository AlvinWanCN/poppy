shell
=========
shell脚本的使用就是对系统命令的使用。

======  =========================================
名称      说明
======  =========================================
$0      脚本名称
$1-9    脚本执行时的参数1到参数9
$?	    脚本的返回值　　　　
$#	    脚本执行时，输入的参数的个数
$@	    输入的参数的具体内容（将输入的参数作为一个多个对象，即是所有参数的一个列表）
$*	    输入的参数的具体内容（将输入的参数作为一个单词）
======  =========================================
